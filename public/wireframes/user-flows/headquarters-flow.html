📊 본사 시스템 플로우 범례

🔹 주요 관리 영역

🏪 점포 관리: 신규 점포 승인부터 운영 관리까지
📋 발주 승인: 점포 발주 요청 검토 및 승인 프로세스
📦 상품 관리: 상품 카탈로그 및 가격 정책 관리
📊 통계 분석: 전체 네트워크 성과 분석 및 인사이트
👥 사용자 관리: 점주, 고객, 관리자 계정 관리
⚙️ 시스템 설정: 전체 시스템 운영 설정 관리

🎯 핵심 승인 프로세스

신규 점포: 신청서 검토 → 승인/거절 → 계정 생성 → 개설 지원
발주 요청: 요청서 검토 → 승인/수정요청 → 발주서 생성 → 배송 관리
상품 등록: 정보 입력 → 검증 → 카탈로그 업데이트 → 점포 알림
점포 제재: 위반 확인 → 제재 조치 결정 → 조치 시행 → 모니터링
