📊 점주 업무 플로우 범례

🔹 주요 업무 영역

📋 주문 관리: 신규 주문부터 완료까지의 전체 프로세스
📦 재고 관리: 재고 조회, 조정, 발주 요청, 입고 처리
📊 매출 분석: 다양한 관점의 매출 데이터 분석
⚙️ 점포 설정: 점포 운영에 필요한 각종 설정 관리

🎯 핵심 업무 플로우

1. 주문 처리: 알림 → 확인 → 수락/거절 → 준비 → 완료
2. 재고 관리: 현황 확인 → 조정/발주 → 입고 → 업데이트
3. 분석 업무: 데이터 조회 → 분석 → 리포트 → 인사이트
4. 설정 관리: 메뉴 선택 → 수정 → 저장 → 적용

🎭 점주 주요 업무 시나리오

🌅 매장 오픈 시 일일 업무

  • 점주 앱 로그인 및 대시보드 확인
  • 전날 주문 완료 여부 최종 확인
  • 재고 현황 점검 및 부족 상품 확인
  • 운영 시간 및 배달 서비스 상태 확인
  • 직원 출근 확인 및 업무 배정
  • 프로모션 및 할인 상품 설정

⚡ 실시간 주문 처리

  • 신규 주문 알림 수신
  • 주문 내역 및 고객 정보 확인
  • 재고 확인 후 주문 수락/거절 결정
  • 주문 준비 시작 및 상태 업데이트
  • 준비 완료 후 고객 알림 발송
  • 픽업 완료 후 주문 완료 처리

📦 주간 재고 관리

  • 전체 재고 현황 리포트 생성
  • 재고 부족 상품 및 과재고 상품 파악
  • 판매 데이터 기반 발주 계획 수립
  • 본사에 발주 요청서 제출
  • 승인 후 배송 일정 확인
  • 입고 시 재고 업데이트 및 검수

📊 월말 매출 분석

  • 월간 매출 현황 종합 분석
  • 인기 상품 및 부진 상품 파악
  • 시간대별 매출 패턴 분석
  • 고객 구매 패턴 및 재방문율 확인
  • 전월 대비 성장률 및 개선점 도출
  • 다음 달 운영 전략 수립

⚠️ 긴급 상황 대응

  • 시스템 오류 또는 네트워크 장애
  • 대량 주문 취소 요청 처리
  • 상품 품질 이슈 발생 시 대응
  • 직원 응급 상황 대체 인력 배치
  • 기계 고장 시 임시 운영 방안
  • 고객 컴플레인 처리 및 보상

⚙️ 점포 설정 및 최적화

  • 계절별 운영 시간 조정
  • 배달 서비스 지역 및 요금 수정
  • 신규 직원 등록 및 권한 설정
  • 알림 설정 개인화 및 최적화
  • 프로모션 및 할인 정책 설정
  • 고객 리뷰 및 피드백 관리
