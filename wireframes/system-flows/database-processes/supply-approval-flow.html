// 물류 요청 생성 및 검증 class SupplyRequestValidator { async validateSupplyRequest(request: SupplyRequestInput): Promise { const validationErrors = []; // 점주 권한 확인 const storeOwner = await this.validateStoreOwner(request.requestedBy, request.storeId); if (!storeOwner.valid) { validationErrors.push('점주 권한이 없습니다'); } // 상품 검증 const productValidation = await this.validateProducts(request.items); if (!productValidation.valid) { validationErrors.push(...productValidation.errors); } // 최소 주문 금액 확인 const totalAmount = request.items.reduce((sum, item) => sum + (item.quantity * item.unitCost), 0 ); const minOrderAmount = await this.getMinOrderAmount(request.storeId); if (totalAmount < minOrderAmount) { validationErrors.push(`최소 주문금액 ${minOrderAmount}원 미달`); } return { valid: validationErrors.length === 0, errors: validationErrors, totalAmount, validatedItems: productValidation.validItems }; } private async validateProducts(items: SupplyRequestItem[]): Promise { const validItems = []; const errors = []; for (const item of items) { const product = await Product.findById(item.productId); if (!product) { errors.push(`상품 ID ${item.productId}를 찾을 수 없습니다`); continue; } if (!product.isActive || !product.isSupplyAvailable) { errors.push(`${product.name}은 공급 불가능한 상품입니다`); continue; } if (item.quantity <= 0 || item.quantity > product.maxOrderQuantity) { errors.push(`${product.name}의 주문 수량이 유효하지 않습니다`); continue; } validItems.push({ ...item, productName: product.name, unitCost: product.costPrice, totalCost: item.quantity * product.costPrice }); } return { valid: errors.length === 0, errors, validItems }; } }

// 본사 승인 처리 시스템 class SupplyRequestApprovalWorkflow { async processApproval(requestId: string, reviewerId: string, decision: ApprovalDecision) { const supplyRequest = await SupplyRequest.findById(requestId); if (!supplyRequest || supplyRequest.status !== 'pending') { throw new Error('승인 가능한 요청이 아닙니다'); } // 본사 관리자 권한 확인 const reviewer = await this.validateHQManager(reviewerId); if (!reviewer.valid) { throw new UnauthorizedError('본사 관리자 권한이 필요합니다'); } switch (decision.action) { case 'approve': return await this.approveRequest(supplyRequest, decision, reviewerId); case 'reject': return await this.rejectRequest(supplyRequest, decision, reviewerId); case 'modify': return await this.modifyRequest(supplyRequest, decision, reviewerId); } } private async approveRequest( request: SupplyRequest, decision: ApprovalDecision, reviewerId: string ) { // 창고 재고 재확인 const stockValidation = await this.validateWarehouseStock(request.items); if (!stockValidation.valid) { throw new InsufficientStockError('창고 재고 부족'); } // 배송 스케줄링 const deliverySchedule = await this.scheduleDelivery( request.storeId, request.requestedDeliveryDate, stockValidation.totalWeight, stockValidation.totalVolume ); // 승인 처리 const trx = await db.transaction(); try { // 요청 상태 업데이트 await trx('supply_requests') .where('id', request.id) .update({ status: 'approved', approved_by: reviewerId, approved_at: new Date(), approval_notes: decision.notes, estimated_delivery_date: deliverySchedule.estimatedDate }); // 승인된 수량 업데이트 (수정사항 반영) for (const item of decision.approvedItems) { await trx('supply_request_items') .where({ supply_request_id: request.id, product_id: item.productId }) .update({ approved_quantity: item.approvedQuantity, notes: item.notes }); } // 배송 정보 생성 const shipment = await this.createShipment(request, deliverySchedule, trx); await trx.commit(); // 알림 전송 await this.notifyApproval(request, shipment); return { approved: true, shipmentId: shipment.id }; } catch (error) { await trx.rollback(); throw error; } } }

// 창고 재고 확인 및 예약 class WarehouseInventoryManager { async checkAvailability(items: SupplyRequestItem[]): Promise { const availabilityResults = await Promise.all( items.map(async (item) => { const warehouseStock = await this.getWarehouseStock(item.productId); const reservedStock = await this.getReservedStock(item.productId); const availableStock = warehouseStock.quantity - reservedStock.quantity; return { productId: item.productId, requestedQuantity: item.quantity, availableQuantity: availableStock, canFulfill: availableStock >= item.quantity, maxAvailable: availableStock, estimatedRestockDate: warehouseStock.estimatedRestockDate }; }) ); const fullyAvailable = availabilityResults.every(result => result.canFulfill); const partiallyAvailable = availabilityResults.some(result => result.maxAvailable > 0 && !result.canFulfill ); return { fullyAvailable, partiallyAvailable, items: availabilityResults, recommendations: this.generateStockRecommendations(availabilityResults) }; } async reserveStock(approvedItems: ApprovedItem[], shipmentId: string): Promise { const trx = await db.transaction(); try { for (const item of approvedItems) { // 창고 재고에서 차감 await trx('warehouse_inventory') .where('product_id', item.productId) .decrement('available_quantity', item.approvedQuantity); // 예약 기록 생성 await trx('warehouse_reservations').insert({ shipment_id: shipmentId, product_id: item.productId, reserved_quantity: item.approvedQuantity, reserved_at: new Date(), reserved_by: 'system' }); // 재고 거래 로그 await trx('warehouse_transactions').insert({ product_id: item.productId, transaction_type: 'reserved_for_shipment', quantity_change: -item.approvedQuantity, reference_id: shipmentId, notes: `물류 요청 배송을 위한 재고 예약` }); } await trx.commit(); } catch (error) { await trx.rollback(); throw error; } } private generateStockRecommendations(results: StockResult[]): Recommendation[] { const recommendations = []; const outOfStock = results.filter(r => r.maxAvailable === 0); if (outOfStock.length > 0) { recommendations.push({ type: 'out_of_stock', priority: 'high', message: `${outOfStock.length}개 상품이 품절 상태입니다`, products: outOfStock.map(r => r.productId) }); } const lowStock = results.filter(r => r.maxAvailable > 0 && r.maxAvailable < r.requestedQuantity ); if (lowStock.length > 0) { recommendations.push({ type: 'partial_fulfillment', priority: 'medium', message: `${lowStock.length}개 상품을 부분 공급만 가능합니다`, alternatives: this.suggestAlternatives(lowStock) }); } return recommendations; } }

// 스마트 배송 스케줄러 class DeliveryScheduler { async scheduleOptimalDelivery( storeId: string, requestedDate: Date, cargo: CargoInfo ): Promise { // 점포 위치 및 배송 정보 조회 const store = await Store.findById(storeId); const deliveryZone = await this.getDeliveryZone(store.location); // 사용 가능한 운송 수단 확인 const availableCarriers = await this.getAvailableCarriers( deliveryZone, cargo.weight, cargo.volume, requestedDate ); if (availableCarriers.length === 0) { // 대안 날짜 제시 const alternativeDates = await this.suggestAlternativeDates( deliveryZone, cargo, requestedDate, 7 // 7일 범위 ); return { canDeliverOnRequestedDate: false, alternatives: alternativeDates, reason: 'no_carrier_available' }; } // 최적 배송 옵션 선택 const optimalCarrier = this.selectOptimalCarrier(availableCarriers, { priorityFactors: ['cost', 'speed', 'reliability'], storePreferences: store.deliveryPreferences }); // 배송 경로 최적화 const route = await this.optimizeRoute( optimalCarrier.depot, store.location, optimalCarrier.existingDeliveries ); return { canDeliverOnRequestedDate: true, carrier: optimalCarrier, estimatedDeliveryTime: route.estimatedArrival, deliveryCost: this.calculateDeliveryCost(optimalCarrier, route), route: route, trackingInfo: await this.generateTrackingInfo(optimalCarrier) }; } private selectOptimalCarrier( carriers: Carrier[], criteria: SelectionCriteria ): Carrier { // 다중 기준 의사결정 알고리즘 (MCDM) const scoredCarriers = carriers.map(carrier => { const scores = { cost: this.normalizeCostScore(carrier.costPerKm), speed: this.normalizeSpeedScore(carrier.averageSpeed), reliability: carrier.reliabilityScore, capacity: this.normalizeCapacityScore(carrier.availableCapacity) }; // 가중 평균 계산 const weightedScore = scores.cost * criteria.weights.cost + scores.speed * criteria.weights.speed + scores.reliability * criteria.weights.reliability + scores.capacity * criteria.weights.capacity; return { carrier, score: weightedScore }; }); // 최고 점수 운송업체 선택 return scoredCarriers .sort((a, b) => b.score - a.score)[0] .carrier; } }
