화면 ID: DEMO-001
메뉴 위치: 시스템 데모 > 모달

화면 개요:
- 편의점 솔루션의 전체 워크플로우를 시각적으로 보여주는 인터랙티브 데모 모달

UI 구성요소:
| 항목명           | 타입    | 필수 | 설명                                      |
|------------------|---------|------|-------------------------------------------|
| 모달 헤더        | Header  | Y    | 제목, 설명, 닫기 버튼                     |
| 탭 네비게이션    | Section | Y    | 주문처리/재고관리/분석리포팅/사용자관리 탭 |
| 컨트롤 버튼      | Section | Y    | 재생/일시정지, 처음부터, 진행률 표시       |
| 워크플로우 시각화| Section | Y    | 단계별 노드와 연결선이 있는 플로우 차트    |
| 단계 노드        | Section | Y    | 역할별 색상의 원형 아이콘과 단계명 표시    |
| 연결선           | Section | Y    | 단계간 흐름을 보여주는 애니메이션 선       |
| 현재 단계 정보   | Section | Y    | 선택된 단계의 상세 설명 패널              |
| 진행 상황 리스트 | Section | Y    | 전체 단계 목록과 완료 상태 표시           |
| 역할 범례        | Section | Y    | 고객/점주/본사 역할 구분 색상 안내        |

버튼 및 액션:
- [닫기 (X)]: 데모 모달 닫기
- [주문처리/재고관리/분석리포팅/사용자관리]: 해당 탭의 워크플로우로 전환
- [재생/일시정지]: 2초 간격 자동 단계 진행 제어
- [처음부터]: 현재 탭의 첫 단계로 초기화
- 각 단계 노드는 시각적 효과로 진행 상황 표시

입출력 데이터:
입력
- 탭 선택, 재생/일시정지/리셋 제어

출력
- 주문처리: 고객주문 → 결제 → 접수 → 준비 → 완료 → 픽업 (6단계)
- 재고관리: 재고확인 → 부족알림 → 물류요청 → 검토 → 배송 → 업데이트 (6단계)
- 분석리포팅: 데이터수집 → 일일요약 → 상품분석 → 통합 → 인사이트 → 권장사항 (6단계)
- 사용자관리: 사용자등록 → 역할할당 → 지점승인 → 접근권한 → 모니터링 → 계정관리 (6단계)
