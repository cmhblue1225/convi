화면 ID: SUP-001
메뉴 위치: 지원센터 > 고객센터
화면 개요: 편의점 솔루션 사용자들에게 FAQ, 문의 접수, 이용가이드를 제공하는 통합 고객 지원 페이지

UI 구성요소:

| 항목명 | 타입 | 필수 여부 | 설명 |
| --- | --- | --- | --- |
| 페이지 제목 | Header | Y | "고객센터" 제목과 설명 텍스트 표시 |
| 탭 네비게이션 | Section | Y | FAQ/문의하기/이용가이드 탭 선택 |
| FAQ 아코디언 | Section | N | 질문 클릭 시 답변 확장/축소 |
| 문의 폼 - 이름 | Text | Y | 문의자 이름 입력 (contact 탭) |
| 문의 폼 - 이메일 | Email | Y | 문의자 이메일 입력 (contact 탭) |
| 문의 폼 - 유형 | Dropdown | Y | 계정/주문/기술/제안/기타 선택 (contact 탭) |
| 문의 폼 - 내용 | Textarea | Y | 문의 내용 입력, 6행 (contact 탭) |
| 이용가이드 카드 | Section | N | 고객/점주/본사별 가이드 표시 (guide 탭) |
| 추가 도움 안내 | Section | N | 추가 지원 안내 메시지 (guide 탭) |

버튼 및 액션:
* [자주 묻는 질문]: FAQ 탭 활성화하여 5개 FAQ 항목 표시
* [문의하기]: 문의 접수 폼 탭 활성화 
* [이용가이드]: 역할별(고객/점주/본사) 이용가이드 카드 표시
* [FAQ 질문 클릭]: 해당 답변 아코디언 확장/축소
* [문의 접수]: 폼 유효성 검사 후 "문의가 접수되었습니다" 알림 표시, 폼 초기화

입출력 데이터:
* 입력: 이름, 이메일, 문의유형, 문의내용 (문의하기 탭에서만)
* 출력: 
  * FAQ 5개 항목 (주문방법, 결제방법, 주문취소, 환불처리, 비밀번호 찾기)
  * 문의 접수 완료 메시지
  * 역할별 이용가이드 정보 (고객/점주/본사 각 5개 항목)
  * 추가 지원 안내 메시지
