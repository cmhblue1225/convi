# 포인트 시스템 구현 완료 보고서

## 1. 구현 완료된 컴포넌트들

### ✅ pointStore.ts (완성)
- **위치**: `src/stores/pointStore.ts`
- **기능**: 포인트 적립, 사용, 회수의 전역 상태 관리
- **주요 함수**:
  - `earnPoints`: 주문 완료 시 포인트 적립 (주문 금액의 1%)
  - `usePoints`: 주문 시 포인트 사용
  - `refundPoints`: 환불 시 포인트 회수 (비례 계산)
  - `fetchUserPoints`: 사용자 포인트 조회
  - `getPointBalance`: 포인트 잔액 조회

### ✅ usePointsValidation.ts (완성)
- **위치**: `src/hooks/usePointsValidation.ts`
- **기능**: 포인트 사용 유효성 검증
- **검증 항목**:
  - 포인트 보유량 확인
  - 주문 금액 대비 사용 한도 (100%)
  - 쿠폰 할인 적용 후 최종 금액 계산

### ✅ types/common.ts (완성)
- **포인트 관련 타입 정의**:
  ```typescript
  export interface Point {
    id: string;
    user_id: string;
    amount: number;
    type: 'earned' | 'used' | 'expired' | 'bonus';
    description?: string | null;
    order_id?: string | null;
    expires_at?: string | null;
    created_at: string;
  }
  ```

## 2. 포인트 적립 로직

### 주문 완료 시 적립
```typescript
// 주문 금액의 1% 적립 (최소 100원)
const pointsToEarn = Math.floor(amount * 0.01);

// 1년 후 만료
const expiresAt = new Date(Date.now() + 365 * 24 * 60 * 60 * 1000);
```

### 적립 조건
- ✅ 주문 완료 상태
- ✅ 로그인된 사용자
- ✅ 주문 금액 > 0
- ✅ 중복 적립 방지 (order_id 기준)

## 3. 포인트 사용 로직

### 사용 규칙
- ✅ 최소 사용 단위: 100포인트
- ✅ 보유 포인트 내에서만 사용
- ✅ 주문 금액을 초과할 수 없음
- ✅ 쿠폰 할인 적용 후 계산

### 유효성 검증
```typescript
const isValidPointsUsage = (points: number): boolean => {
  if (points < 0) return false;
  if (points > totalPoints) return false;
  if (points > maxUsablePoints) return false;
  return true;
};
```

## 4. 포인트 회수 로직 (환불 시)

### 회수 계산 방식
```typescript
// 환불 금액에 비례하여 포인트 회수
const pointsToRefund = Math.floor(
  (refundAmount / orderAmount) * totalEarnedPoints
);
```

### 회수 프로세스
1. 해당 주문으로 적립된 포인트 조회
2. 환불 금액 비례 계산
3. 포인트 차감 및 거래 내역 기록
4. 로컬 상태 업데이트

### 회수 시나리오
- **전액 환불**: 적립된 포인트 100% 회수
- **부분 환불**: 환불 금액 비례하여 포인트 회수
- **포인트 미사용**: 회수할 포인트 없음

## 5. 데이터베이스 연동

### RPC 함수 호출
```typescript
// 포인트 적립
const { data } = await supabase.rpc('earn_points', {
  user_uuid: userId,
  point_amount: pointsToEarn,
  point_description: description,
  order_uuid: orderId
});

// 포인트 사용
const { data } = await supabase.rpc('use_points', {
  user_uuid: userId,
  point_amount: amount,
  point_description: description,
  order_uuid: orderId
});

// 포인트 회수
const { data } = await supabase.rpc('refund_points', {
  user_uuid: userId,
  order_uuid: orderId,
  point_amount: pointsToRefund,
  point_description: description
});
```
